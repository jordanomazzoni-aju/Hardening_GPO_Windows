# Windows - Hardening Básico - Part 77 - Toast notifications - Help Experience

Garanta que 'Turn off toast notifications on the lock screen' esteja atribuido como 'Enabled'

Essa configuração desativa as notificações pop-up na tela de bloqueio.

O estado recomendado para essa configuração é Ativado.

Embora esse recurso possa ser útil para os usuários, aplicativos que fornecem notificações pop-up podem exibir dados pessoais ou comerciais confidenciais enquanto o dispositivo estiver sem supervisão.

User Configuration\Policies\Administrative Templates\Start Menu and Taskbar\Notifications\Turn off toast notifications on the lock screen

Garanta que 'Turn off Help Experience Improvement Program' esteja atribuido como 'Enabled'

Essa configuração de política especifica se os usuários podem participar do programa de Aprimoramento da Experiência de Ajuda. O programa de Aprimoramento da Experiência de Ajuda coleta informações sobre como os clientes usam a Ajuda do Windows para que a Microsoft possa melhorá-la.

O estado recomendado para essa configuração é: Ativado.

Em grandes ambientes corporativos gerenciados, pode não ser desejável que a Microsoft colete informações de computadores cliente gerenciados.

User Configuration\Policies\Administrative Templates\System\Internet Communication Management\Internet Communication Settings\Turn off Help Experience Improvement Program
