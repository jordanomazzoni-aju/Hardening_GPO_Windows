# Hardening_GPO_Windows

Este repositório centraliza políticas de grupo (GPO) essenciais para o endurecimento (hardening) de ambientes Windows. 

Voltado para administradores de sistemas e profissionais de segurança, o projeto oferece configurações prontas para mitigar vulnerabilidades comuns, garantir conformidade com benchmarks de segurança (como CIS ou STIG) e proteger estações de trabalho e servidores contra ameaças cibernéticas. 

É um recurso prático para quem busca elevar a postura de segurança de infraestruturas Microsoft de forma padronizada.
