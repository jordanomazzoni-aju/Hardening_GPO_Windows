# Windows - Hardening Básico - Part 58 - Microsoft Support Diagnostic Tool

Garanta que 'Microsoft Support Diagnostic Tool: Turn on MSDT interactive communication with support provider' esteja configurado como 'Disabled'

Esta configuração de política configura a comunicação interativa da Ferramenta de Diagnóstico de Suporte da Microsoft (MSDT) com o provedor de suporte. A MSDT coleta dados de diagnóstico para análise por profissionais de suporte.

O estado recomendado para esta configuração é: Desativada.

Em ambientes de alta segurança, os dados nunca devem ser compartilhados com terceiros sem consentimento explícito, pois podem conter informações confidenciais.
